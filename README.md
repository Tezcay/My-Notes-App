## 📖 1. 项目简介

这是一个基于 **原生 Web 技术（HTML5 + CSS3 + JavaScript）** 开发的个人笔记管理系统。  
项目界面在布局与交互上参考并复刻了 **「小米云服务｜笔记」** 的设计风格，旨在提供一个 **简洁、高效、易用** 的笔记记录体验。

本项目严格遵循课程任务书要求：

- ✅ **零依赖**：不使用任何 CSS 框架（如 Bootstrap）或 JavaScript 框架（如 Vue / React）
- ✅ **纯原生实现**：所有交互逻辑均基于原生 DOM API
- ✅ **数据持久化**：使用 `window.localStorage` 保存数据，刷新页面不丢失

## ✨ 2. 功能说明

### 2.1 核心功能

- ✅ **三栏式布局**  
  采用经典的「分类栏 - 笔记列表 - 编辑区」三栏结构，界面清晰，逻辑明确

- ✅ **笔记 CRUD 功能**
  - 新建笔记
  - 查看笔记
  - 编辑笔记（实时保存）
  - 删除笔记

- ✅ **分类管理**
  - 支持新建自定义文件夹
  - 支持删除文件夹  
  - **级联处理逻辑**：删除文件夹后，其下笔记会自动移动至「未分类」

- ✅ **搜索功能**
  - 支持根据 **标题或内容关键词** 实时搜索笔记

- ✅ **交互细节优化**
  - 分类折叠 / 展开
  - 当前选中项高亮显示
  - 悬浮添加按钮

## 📖 3. 使用指南（User Guide）

### 3.1 基础操作

1. **新建笔记**  
   点击中间笔记列表栏右下角的黄色 `+` 悬浮按钮，即可创建新笔记

2. **编辑笔记**  
   在右侧编辑区直接修改笔记标题或内容，系统会自动实时保存

3. **删除笔记**  
   选中笔记后，点击编辑区右上角的 🗑️ 垃圾桶按钮即可删除

---

### 3.2 高级操作（分类管理）

1. **新建文件夹**  
   点击左侧「我的文件夹」右侧的 `+` 按钮，输入名称后即可创建

2. **删除文件夹**  
   在左侧文件夹名称上 **点击鼠标右键**，确认后即可删除  

   注意：文件夹删除后，其中的笔记不会丢失，而是会自动移动到「未分类」

3. **折叠 / 展开文件夹列表**  
   点击「我的文件夹」左侧的小箭头即可控制展开状态

### 4.2 数据结构设计

本项目通过 **两个核心数组对象** 管理数据，并统一同步至 `LocalStorage`。

#### 1️⃣ 笔记数据（notes）

    {
      id: 1734756210234,          // 时间戳作为唯一 ID
      title: "示例笔记",
      content: "笔记正文内容...",
      updateTime: "刚刚",
      categoryId: "folder-1"      // 所属分类 ID
    }

#### 2️⃣ 分类数据（categories）

    {
      id: "folder-1734756220111", // 分类唯一 ID
      name: "学习资料"            // 分类名称
    }

---

### 4.3 核心函数说明

- **renderNoteList()**  
  根据当前选中的 `currentCategoryId`，筛选 `notes` 数组并动态渲染笔记列表

- **renderFolderList()**  
  遍历 `categories` 数组，生成左侧分类栏，并绑定右键删除事件

- **handleDeleteCategory(category)**  
  删除分类的核心逻辑：  
  1. 遍历所有笔记  
  2. 将属于该分类的笔记 `categoryId` 重置为 `'uncategorized'`  
  3. 从分类数组中移除该分类  

- **saveAllData()**  
  对 `localStorage.setItem()` 进行统一封装，确保数据修改时的一致性与可靠性
