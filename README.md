# 📘 个人笔记管理系统 (Web Note App)

> 《Web 程序设计实践》课程期末项目 - 2023级软工

## 📖 项目简介

这是一个基于原生 Web 技术（HTML5 + CSS3 + JavaScript）开发的个人笔记管理系统。项目界面深度复刻了“小米云服务|笔记”的风格，旨在提供清爽、高效的笔记记录体验。

本项目完全遵循课程任务书要求，**不依赖任何现有的 CSS 框架（如 Bootstrap）或 JS 框架（如 Vue/React）**，所有交互与数据管理均通过原生 DOM 操作和 LocalStorage 实现。

## 🛠 技术栈

* **结构**: HTML5 (语义化标签)
* **样式**: CSS3 (Flexbox 布局, 纯手写样式, FontAwesome 图标库)
* **逻辑**: Native JavaScript (ES6+)
* **存储**: LocalStorage (浏览器本地存储)

## ✨ 功能特性

### 1. 界面与交互
* **三栏式布局**：经典的“分类 - 列表 - 编辑”三栏设计，层级清晰。
* **像素级复刻**：精美的 UI 细节，包括小米风格的配色、圆角搜索框、悬浮新建按钮等。
* **响应式设计**：适配不同屏幕尺寸。

### 2. 核心功能 (开发中)
* **分类管理**：支持“全部笔记”、“未分类”及自定义文件夹管理。
* **笔记管理**：支持笔记的新建、编辑、删除。
* **实时保存**：利用 LocalStorage 实现数据持久化，刷新页面不丢失。
* **富文本编辑**：提供加粗、标题、列表等基础编辑工具。
* **搜索功能**：支持按标题或内容关键词快速检索笔记。
