---

## 📖 1. 项目简介

这是一个基于 **原生 Web 技术（HTML5 + CSS3 + JavaScript）** 开发的现代化个人笔记管理系统。

项目界面深度复刻了 **「小米笔记」** 的清爽风格，并在此基础上实现了 **响应式布局、夜间模式、拖拽归档** 等高级交互功能，旨在提供极致的 Web 笔记体验。

**核心技术亮点：**

* ✅ **零依赖**：无 Vue/React，无 Bootstrap，纯原生 JavaScript 手写核心逻辑。
* ✅ **响应式设计**：一套代码完美适配 PC 端与移动端设备。
* ✅ **数据持久化**：基于 `LocalStorage` 实现数据实时保存与状态记忆。

## ✨ 2. 功能特性 (Features)

### 🎨 2.1 界面与体验

* **三栏式布局**：经典的「文件夹导航 - 笔记列表 - 编辑区域」设计。
* **🌗 夜间模式 (Dark Mode)**：
* 支持一键切换深色/浅色主题。
* 自动记忆用户偏好，刷新页面不失效。


* **📱 移动端适配 (Responsive)**：
* 屏幕宽度小于 768px 时自动切换为移动视图。
* 侧边栏改为抽屉式交互，编辑器采用滑入式覆盖体验。


* **🔍 智能搜索高亮**：
* 全文检索笔记标题与内容，匹配关键词自动**高亮显示**（绿色加粗），快速定位。



### 🛠 2.2 核心交互

* **📝 笔记管理**：
* 支持新建、编辑（实时保存）、删除笔记。
* **回收站机制**：普通删除移入“最近删除”，在回收站中再次删除则永久销毁。


* **📂 文件夹系统**：
* 支持新建自定义文件夹，右键可删除文件夹（级联逻辑：文件夹删除后，其下笔记自动归入“未分类”）。


* **🖱️ 拖拽归档 (Drag & Drop)**：
* 支持直接将笔记从列表中**拖拽**到左侧文件夹进行快速分类。


* **🔃 多维排序**：
* 支持按“编辑时间倒序”、“编辑时间正序”、“标题 A-Z”三种模式切换。

## 📖 3. 使用指南 (User Guide)

### 3.1 基础操作

1. **新建笔记**：点击列表栏右下角的黄色 `+` 悬浮按钮。
2. **切换视图**：
* **PC端**：点击左侧导航栏切换分类。
* **手机端**：点击左上角 `≡` 菜单键呼出侧边栏。


3. **切换主题**：点击编辑器右上角的 🌙/☀️ 图标切换日夜模式。
4. **排序切换**：点击列表上方的“按编辑时间”文字，循环切换排序方式。

### 3.2 进阶操作

* **拖拽整理**：按住笔记列表中的任意项，将其拖动到左侧的文件夹上，松手即可完成分类移动。
* **私密笔记**：(界面入口已预留，功能开发中)

## 🛠️ 4. 开发文档 (Developer Documentation)

### 4.1 目录结构

```text
MyNotesApp/
├── index.html          # 页面骨架 (语义化 HTML5)
├── styles/
│   └── style.css       # 样式表 (CSS Variables, Flexbox, Media Queries)
├── scripts/
│   └── script.js       # 核心逻辑 (DOM 操作, 事件委托, LocalStorage)
└── README.md           # 项目文档

```

### 4.2 数据结构 (LocalStorage)

**1. 笔记数据 (`notes`)**

```javascript
{
  id: 1735205810234,        // 唯一ID (时间戳)
  title: "学习计划",
  content: "复习 JavaScript...",
  updateTime: 1735205899999,// 最后编辑时间
  categoryId: "folder-1"    // 所属分类ID
}

```

**2. 分类数据 (`categories`)**

```javascript
{
  id: "folder-1735205822222",
  name: "前端开发"
}

```

### 4.3 关键技术实现

* **主题切换**：通过 JS 切换 `<html>` 标签的 `data-theme="dark"` 属性，配合 CSS 变量 (`var(--bg-primary)`) 实现全局即时换肤。
* **拖拽 API**：监听 `dragstart`, `dragover`, `drop` 事件，通过 `dataTransfer` 传递笔记 ID 实现跨区域交互。
* **防抖搜索**：监听输入框 `input` 事件，实时过滤数组并重新渲染 DOM，配合 `RegExp` 实现关键词高亮替换。
